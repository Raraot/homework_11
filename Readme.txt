Додана логіка у цьому ДЗ:
- щоб додати день народження вводимо для існуючого контакту:  birthday Fedir 25-07-1985 (birthday name date).
- щоб дізнатись кількість днів до наступного дня народження:  left Fedir (left name).
- команда show all реалізує метод iterator, який за одну ітерацію повертає частину телефонної книжки із трьох контактів, 
для того щоб вивести наступні сторінки телефонної книжки потрібно ввести комаду next.
- введена setter та getter логіка для value класу Phone.
- організована перевірка corrector_birthday та corrector_phone правильності вводу телефонного номеру та дати дня народження.

ВЕСЬ СПИСОК КОМАНД доданий в минулих ДЗ
- вітання з CLI-ботом                        - hello
- додавання контакта з номером телефону      - add name tel.      (example: add Volodymyr 345-45-45)
- повторна команда "add name tel." додасть ще один номер телефону контакту
- зміна номера телефону для контакту         - change name tel.   (example: change Volodymyr 2345789)
- читання номерів телефонів контакту         - phone name         (example: phone Volodymyr)
- показати усю телефонну книгу по-сторінково - show all
- вийти із виконання програми                - stop               (або "good bye", "close", "exit", "stop",)
